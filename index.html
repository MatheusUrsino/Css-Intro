CSS - Introdução

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Ele permite que os desenvolvedores controlem a aparência e o layout das páginas da web, separando o conteúdo da apresentação.

Como funciona o CSS?

O CSS funciona aplicando regras de estilo aos elementos HTML. Essas regras são compostas por seletores, propriedades e valores. Os seletores identificam quais elementos HTML serão estilizados, enquanto as propriedades e valores definem como esses elementos serão apresentados.

Nele, podemos definir:

  • Cor do texto
  • Fonte e tamanho
  • Layout e posicionamento
  • Estilos de borda e fundo

Existem três formas de aplicar o CSS:

  1. Inline: aplicando estilos diretamente aos elementos HTML
  2. Internal: definindo estilos dentro da tag <style> no cabeçalho do documento
  3. External: utilizando um arquivo separado com extensão .css e vinculando-o ao documento HTML

como referenciar o CSS Externo

Para referenciar um arquivo CSS externo, utilize a tag <link> no cabeçalho do documento HTML:

<link rel="stylesheet" href="estilo.css">
